## 什么是对象

在javascript中，可以说一切皆对象。

可以使用两种形式来创建对象: 声明形式和构建形式。

```js
// 声明形式
var  obj = {
  key: 1,
}

// 构建形式
var obj2 = new Object();
obj2.key = 1;
```
