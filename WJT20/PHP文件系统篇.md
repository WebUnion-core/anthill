## 查询API ##

### basename ###

功能: 返回路径中的文件名部分:

语法规则:

```
string basename (string $path [, string $suffix ])
```

1. path 参数是一个文件路径字符串，注意，Windows 系统允许使用斜杠反斜杠作为目录分隔符，而其他系统环境只允许使用斜杠，所以为了兼容系统，应避免使用反斜杠;
